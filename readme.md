## ⚙️ How It Works

### 🔹 UI & Layout

* Built using modern CSS techniques (Grid + Flexbox)
* Uses CSS variables for theme switching

### 🔹 Theme Toggle

* JavaScript switches between light and dark mode dynamically

### 🔹 Animations

* Implemented using Intersection Observer API
* Elements reveal on scroll smoothly

### 🔹 Contact Form

* Currently connected to Formspree (can be replaced with EmailJS)
* Includes basic validation before submission

---

## ⚠️ Notes

* Make sure the form endpoint (Formspree) is active
* If using EmailJS, replace keys in `main.js`
* Ensure images paths are correct when deploying

## 🚀 Deployment (Vercel)

To deploy the project:

1. Push your code to GitHub
2. Go to 👉 https://vercel.com
3. Import your repository
4. Click **Deploy**

✅ Your website will be live instantly
